# Online Shoppers Purchasing Intention Analytics

This repository contains the ITBAN 3 and ITBAN 4 final project for descriptive and diagnostic analytics of online shopping customer behavior.

## Project title
Descriptive and Diagnostic Analytics of Online Shopping Customer Behavior

## Dataset
- File used: `online_shoppers_intention.csv`
- Original source: UCI Machine Learning Repository - Online Shoppers Purchasing Intention Dataset
- Raw shape: 12,330 rows and 18 columns
- Cleaned shape: 12,205 rows and 24 columns

## Business question
What visitor behavior patterns separate purchasing sessions from non-purchasing sessions, and what factors are associated with higher purchase conversion?

## How to run
1. Install Python 3.10 or later.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the notebook:
   ```bash
   jupyter notebook online_shoppers_analysis.ipynb
   ```
4. Run all cells from top to bottom.
